## What is in this folder?
1.  QA_Predictive Modelling Round 3 (RFC pipeline, Leakage fix).ipynb - third round attempt, using a random forests classifier with a pipeline, identify why the accuracy of the classification remained low. Main results: RFC is the best among all, yet the dataset labels were too subjective as they came from qualitative evaluation (can also be seen in the clusters). The pipeline can be used as a first indicator before any qualitative inspections. 
2. QA_Predictive Modelling Implementation.ipynb - using the pipeline from QA_Predictive Modelling Round 3 script to predict on new dataset. A course table with predicitons and average enrol & issued certificate in the past 6 month  is the final output of this script.
3. QA_Predictive Modelling Round 2.ipynb - second round attempt, using random forests for the classification problem, exploring different feature engineering approaches, yet the results are also not optimal.
4. QA_Predictive Modelling.ipynb - the first round of constructing various classifiers for the classification problem; finding: Decision Tree outperformed baseline classifiers.
