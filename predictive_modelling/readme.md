# About this folder
## What's inside this folder?
There are two folders in this folder:
1. QA Classifier - contains the notebooks for building a multi-class classifier (random forests) for online course review (course quality control).
2. TS Modelling - time series data modelling for KPI prediction, including two algorithms: 1. moving average, 2. scale factor
## Notes
The datasets are extracted from the learning management system (Moodle)
Dependencies: pandas, numpy, matplotlib, seaborn, scikit-learn, datetime, re, math, statsmodels
