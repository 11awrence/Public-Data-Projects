# About this Repo
This repo contains my data projects, demonstrating skills in exploratory data analysis, multimodal data analysis, and predictive modelling for educational data.
## Folder guide
- exploratory_data_analysis - analysis for the course development on Moodle, as well as the course surveys.
- multimodal_learning_analytics - multimodal learning analytics project in 2021 - 2023 (served the Frontier in Education publication)
- predictive_modelling - two predictive modelling projects in this folder - multi-class classification and timeseries prediction
- scripts - analyses that serve specific purposes
