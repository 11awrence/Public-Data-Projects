# About this folder
## What's inside this folder?
There are two folders in this folder:
1. Best of Analysis - serves for the blog post "Top 5 courses in year 20xx"
2. HEI Integration_CleanUp -  user categorisation based on email domain. "Which higher education institution do they come from?"
3. HEI Integration_growth - an extension of the categorisation, adding timestamps so that the changes can be traced in a continuous flow.
4. Moodle Course and Cert merge- merging the output from two SQL views, which is used for Course_development analysis.
## Notes
The datasets are extracted from the learning management systems (Moodle, openHPI)
Dependencies: pandas, numpy, matplotlib, seaborn, re, json
