# About this folder
## What's inside this folder?
There are two notebooks in this folder:
1. Course_development.ipynb - tracks the course development, i.e., number of enrolments, number of issued certificates,  across different months. It also explores the impact of course language and course workload on the course enrolments and issued certificates. The analysis also includes a k-Means clustering. 
2. Moodle_Surveys.ipynb - analysis of the course pre- and post-surveys, extraction of subjective perceived knowledge gain and learner profiles (with k-Mean algorithm)
## Notes
The datasets are extracted from the learning management system (Moodle)
Dependencies: pandas, numpy, matplotlib, seaborn, scikit-learn
